람다식 = method 안에 method가 들어가는 형식 



**this, this(), super, super()**



this : 현재 객체의 모든 멤버 접근 가능 (자신 클래스 가르킴)

super : 현재 객체의 부모 클래스의 멤버 접근 가능 

this() : 내 생성자에서 다른 내 생성자를 호출 할 때 사용

super() : 자식 클래스의 생성자에서 부모 클래스의 생성자를 호출 할 때 사용
